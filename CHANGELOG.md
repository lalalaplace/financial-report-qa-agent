# Changelog

## V1.1.0 - 2026-07-03

新增受控 LLM 结果洞察能力。

### Added

- 新增 `llm_insight_node`，在确定性主答案生成后追加“补充解读”和“可继续分析”。
- 洞察节点只基于 `query_result`、`analysis_result` 和主答案，不参与 QueryPlan、SQL 生成、数值计算和成功失败判断。
- 同比查询补充解释边界和后续分析方向，不复述主答案中的本期值、上期值、同比率和差额。
- 趋势查询补充趋势形态；对比查询补充指标口径边界；排名查询补充数据库样本范围说明。
- 增加真实 LLM smoke 脚本，用于验证点查跳过、同比不复述、趋势洞察、排名口径、空结果跳过和原因分析拒答。

### Changed

- 最终答案改为先生成确定性主答案，再按需拼接 LLM 补充洞察。
- LLM 洞察失败、输出为空或内容越界时不影响主查询答案。

## V1.0.0 - 2026-06-29

首次公开发布版本。

### Included

- 财报 PDF 结构化抽取流程。
- PostgreSQL 财务数据底座。
- 字段级 lineage 与质量校验。
- LangGraph 财报问数 Agent。
- QueryPlan 驱动的受控查询链路。
- 模板 SQL 生成、执行前校验和只读执行器。
- 单点、趋势、同比、派生指标、公司对比、排名和多轮追问能力。
